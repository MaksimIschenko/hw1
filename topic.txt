Команда консоли чтобы создать топик:
```
docker-compose exec kafka-0 kafka-topics.sh --create --topic test-topic-homework-1 --bootstrap-server localhost:9092 --partitions 3 --replication-factor 2
```

Подробная информация о созданном топике:
```
docker-compose exec kafka-0 kafka-topics.sh --describe --topic test-topic-homework-1 --bootstrap-server localhost:9092
```
Информация:
```
Topic: test-topic-homework-1	TopicId: N3_ML9y_S0SOrRszeN0g8A	PartitionCount: 3	ReplicationFactor: 2	Configs: 
	Topic: test-topic-homework-1	Partition: 0	Leader: 2	Replicas: 2,0	Isr: 2,0
	Topic: test-topic-homework-1	Partition: 1	Leader: 0	Replicas: 0,1	Isr: 1,0
	Topic: test-topic-homework-1	Partition: 2	Leader: 1	Replicas: 1,2	Isr: 2,1
```
